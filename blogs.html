JavaScript Question

1.What is the difference between let, const, and var in JavaScript?

a. var: Function-scoped, can be redeclared and reassigned, hoisted with an undefined value.

b. let: Block-scoped, can be reassigned but not redeclared in the same scope, hoisted without initialization.

c. const: Block-scoped, cannot be reassigned or redeclared, must be initialized when declared.

2. What is the difference between find() and filter() method??

a.find(): Returns the first element that satisfies the condition.

b. filter(): Returns an array of all elements that satisfy the condition.

3.How do you select an element in the DOM using JavaScript?

Name a few methods:

a. document.getElementById('id') – Selects an element by its ID.

b. document.getElementsByClassName('class') – Selects elements by class name.

c. document.getElementsByTagName('tag') – Selects elements by tag name.

d. document.querySelector('selector') – Selects the first element matching the CSS selector.

e. document.querySelectorAll('selector') – Selects all elements matching the CSS selector.
